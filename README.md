# 🎮 Water Sort Puzzle — Android

Игра-головоломка "Сортировка шариков". Собирается автоматически через GitHub.

---

## 📋 Пошаговая инструкция

### Шаг 1 — Скачайте gradle-wrapper.jar

Этот файл нужен для сборки. Скачайте его по ссылке и положите в папку
`gradle/wrapper/gradle-wrapper.jar`:

🔗 https://github.com/gradle/gradle/raw/v8.4.0/gradle/wrapper/gradle-wrapper.jar

### Шаг 2 — Создайте репозиторий на GitHub

1. Зайдите на **github.com**
2. Нажмите кнопку **"+"** вверху справа → **"New repository"**
3. Название: `WaterSortPuzzle`
4. Выберите **Public**
5. Нажмите **"Create repository"**

### Шаг 3 — Загрузите файлы

1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите ВСЕ файлы и папки из архива в окно браузера
3. Нажмите **"Commit changes"**

> ⚠️ Важно: папки должны сохранить структуру!
> Загружайте именно папки, а не только файлы внутри.

### Шаг 4 — Дождитесь сборки APK

1. Нажмите вкладку **"Actions"** в вашем репозитории
2. Вы увидите запущенный процесс "Build APK" (жёлтый кружок = идёт сборка)
3. Подождите 3–5 минут
4. Когда появится зелёная галочка ✅ — APK готов!

### Шаг 5 — Скачайте APK

**Способ 1 — через Releases:**
1. Нажмите **"Releases"** на главной странице репозитория (справа)
2. Нажмите на файл `.apk` — он скачается

**Способ 2 — через Actions:**
1. Нажмите **"Actions"** → кликните на последний запуск
2. Внизу страницы найдите **"Artifacts"**
3. Скачайте **"WaterSortPuzzle-APK"**

### Шаг 6 — Установите на телефон

1. Отправьте APK на телефон (через Telegram, Email или USB)
2. Откройте файл на телефоне
3. Если появится предупреждение — нажмите **"Установить всё равно"**
   (это нормально для файлов не из Google Play)
4. Готово — играйте! 🎉

## 🎮 Как играть

- Нажмите на колбу → она подпрыгнет (выбрана)
- Нажмите на другую колбу → шарик переместится (если ход разрешён)
- **Можно** класть шарик: в пустую колбу или на шарик того же цвета
- **Цель**: в каждой колбе — шарики только одного цвета
- **Restart** — начать уровень заново
- **Undo** — отменить последний ход
- **Hint** — подсказка (мигнут нужные колбы)
- **Skip** — пропустить уровень

## ❓ Частые вопросы

**Почему GitHub показывает красный крестик ✗?**
Нажмите на него → посмотрите на сообщение об ошибке → напишите мне, я помогу.

**Телефон не устанавливает APK?**
Зайдите в Настройки → Безопасность → включите "Неизвестные источники" или
"Установка из неизвестных источников".

**Можно ли добавить новые цвета?**
Да! В файле `LevelManager.java` найдите массив `COLORS` и добавьте строку:
```java
Color.parseColor("#FF00FF"), // Розовый
```
